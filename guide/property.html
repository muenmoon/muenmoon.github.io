# IFiltersConfig

参数 说明 类型 是否必填 默认值

title 筛选表单项 label String 是 -

slotName 表单项插槽名，需要和具名插槽名称对应 slot 否 -
rules 表单项校验规则，参考 Element UI String 否 -

itemProps el-form-item 的配置项 Object 否 -
on element 组件事件，直接透传 Object 否 默认会拦截 change及input事件并处理更新数据,如有特殊情况请自定义方法实现
slots element 组件 slots Object 否 格式：{element UI中 的 Slots 名:VNode/String }

# props中自定义属性

TIP

当 element 为el-select、el-radio-group、el-checkbox-group、el-checkbox时，可以配置options作为子选项集，包含的默认属性为label、value，可以通过optionLabel、optionValue修改属性名。

参数 说明 类型 默认值
options 特定 element 时的子选项集 Array<{value, label}> -
optionLabel 指定每个选项对象中作为选项标签或显示文本的属性 String 'label'
optionValue 指定每个选项对象中作为选项值关联的属性 String 'value'

# 事件

事件名 说明 类型 默认值
loadData 接口请求方法 () => void 请求接口数据方法，用于内部处理搜索按钮点击事件
handleSubmit 筛选器查询按钮点击事件 (params) => void 非必填，不填使用内置事件处理。不填需要依赖 loadData 方法
handleReset 筛选器重置按钮点击事件 (params) => void 非必填，不填使用内置事件处理
modalConfirm 弹窗点击确认事件 (params) => Promise params 中包括了表单控件值以及当前操作类型
modalSubmit 弹窗点击提交事件 (params) => Promise params 中包括了表单控件值以及当前操作类型
modalCancel 弹窗点击取消事件 ({crudOperationType}) => void crudOperationType 是指当前的操作类型；
handleCreate 新增按钮点击事件 () => void 按钮文案配置使用 modalconfig 中的 addBtnText
rowDel 操作列删除按钮点击事件 （row）=> void row:当前行数据
rowEdit 操作列编辑按钮点击事件 （row）=> void row:当前行数据
rowView 操作列查看按钮点击事件 （row）=> void row:当前行数据

# 方法

方法名 说明 类型 默认值
switchModalView 操作弹窗显示与关闭 (state, operationType,row) => Promise state:true（打开弹窗）false(关闭弹窗)operationType:操作类型名称，包括默认的 ADD,DELETE,UPDATE,VIEW 以及自定的 ICustomOperationTypes 中的 typeName row:需要传递给弹窗的数据信息，以便在自定义插槽中获取配合使用。 打开弹窗:this.$refs.crud.switchModalView(true,operationType,{...row}); 关闭弹窗：this.$refs.crud.switchModalView(false)
setFormFields 给弹窗里的表单设置值 （values）=> void
getVxeTableRef 获取表格实例 相关操作参考 vxe-table 官方说明
getCheckboxRecords(isFull) 用于 type=checkbox，获取当前已选中的行数据（当前列表，如果 isFull=true 则获取全表已选中的数据） 参考 vxe-table 官方说明
getCheckboxReserveRecords(isFull) 用于 checkbox-config.reserve，获取已保留选中的行数据（不包含当前列表，如果 isFull=true 则不包含全部列表） 参考 vxe-table 官方说明
getCheckboxIndeterminateRecords(isFull) 用于 tree-config 和 type=checkbox，获取半选状态的行数据（当前列表，如果 isFull=true 则获取全表已选中的数据） 参考 vxe-table 官方说明
removeCheckboxRow() 删除复选框选中的行数据 参考 vxe-table 官方说明
clearCheckboxRow 用于 type=checkbox，手动清空用户的选择 参考 vxe-table 官方说明
clearCheckboxReserve 用于 checkbox-config.reserve，手动清空用户保留选中的行数据 参考 vxe-table 官方说明
resetTableTab 重置 el-tabs 选中项，默认选中第一个标签页 (val)=>void
getModalFormRef 获取弹窗表单el-form实例 参考element UI官方说明

# 插槽

插槽名称 说明 插槽 Prop
defaultTitle 标题插槽位，传入插槽则不展示标题
defaultHeader 主按钮左侧区域插槽位，便于多按钮场景扩展。
筛选项(AutoFilters)插槽 透传给筛选(AutoFilters)组件的插槽，名称需和配置项中保持同步。 { item }
弹窗表单项(DynamicForm)插槽 透传给弹窗表单(DynamicForm)组件的插槽，名称需和配置项中保持同步。 { item, params }
表格 VxeTable 组件插槽 透传给 VxeTable 组件的插槽，名称需和配置项中保持同步。 具体 props 参考 vxe-table 官网
pageCenter 筛选组件和表格组件中间区域插槽 无
menu 添加表格操作列自定义按钮 {row,tableRow}，row 为：表格行信息加上+modalConfig.formconfig 配置的字段，相同字段名前者会覆盖后者的值；tableRow 为表格行息
modalDefault 弹窗中的默认插槽 { row, operationType }
modalFooter 弹窗中的底部插槽 { row, operationType }
弹窗自定义插槽，在 modalconfig.customOperationTypes 中定义的插槽名 弹窗中自定义操作对应的插槽 { row, operationType }
filterCustomBtn 查询重置插槽
